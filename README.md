# alpenbrevet
Python script to evaluate your own bike marathon time among Alpenbrevet 2021 participants

# How to
Change your preferences in [start.py](start.py)
```
# Set your preferences here.
strecke = 'gold'
start_nr = 1097
```

Then run the script
```
python3 stats.py
```
